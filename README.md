# PyDNS
Direct Numerical Simulation of Fluid Flow with Direct Forcing IBM Using Python

## Requisites
1. Python3
2. Numpy
3. Scipy
4. Sympy
5. Matplotlib

## Instructions
Recommended Usage with pyenv and venv.
1. Clone repository:
    ```sh
    git clone https://github.com/nauaneed/PyDNS
    ```
2. Install dependencies

3. Run:
    ```sh
    python -m src.PyDNS
    ```
